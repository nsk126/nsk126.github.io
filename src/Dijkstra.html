Dijkstras' Algorithmn

The Dijkstras' Algorithmn can be applied to a certain graph to find the shortest distance between a start node to a goal node. A feature of this Algorithmn is that it can make paths to arbitary graphs with no negative edge-weights

  • The points or nodes in a graph are connected by certain links or edges.
  • Each edge has a set weight and this will determine the cost of node A to node B via edge x
