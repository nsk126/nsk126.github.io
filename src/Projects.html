Procedural level generation for 2D Platformer games: A test using different techniques

2D Platformer games are everything similar to Nintendos' 1985 game, Super Mario Bros. With a simple character that moves on your command with objects to traverse on and a very simple implementation of physics engine, I made a very simple skeletal 2D game and intend to test all possible ways to build a procedural level generation model or atleast mimic the effects of a well-used level generation technique called Perlin Noise. Electrical and Control Design of an Autonomous Underwater Vehicle(AUV)

To build the first AUV in my home state and compete in an International AUV competition( SAUVC-2020 ), I along with a few students in my undergrad founded a group called AUVMEC . I lead the Electrical & Controls team and built the entire Electronics & Software stack, from low-level C program firmware running on the ATmega2560, to a behavior model running on a NVDIA Jetson Nano. I also desinged a PCB for a simple power and signal distribution system and deployed algorithms like the Kalman Filter for sensor noise on board and an LQR controller implementation for movement.

Dynamic roll and pitch self-correcting Stewart Platform

I built a simple Gough-Stewart platform with means of 6 linear actuators, 2 Laser cut acrylic platforms and universial joints. Used a Linear Kalman filter to find accurate roll and pitch angle data from the noisy IMU readings. Modelled inverse kinematics using simple rotational matrices to get a fully tilt self-correcting stewart platform.

An Extended Kalman Filter implementation to estimate the State of Charge(SoC) of a Battery

To get very accurate estimates of the State of Charge of Lithium-Ion batteries, using moderate quality sensors which may harbor noisy current measurements, an Extended Kalman Filter is implemented on a Microcontroller that measures current and voltage readings in real-time and estimates the battery SoC with a model of the battery dynamics.
